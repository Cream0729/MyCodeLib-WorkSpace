HMTL元素分类：

  1. 块元素(可设置宽高，独立成行)：h1~6，p，div，ul，li。
  2. 行内元素(不可设置宽高，不独立成行)：a，span。
  3. 行内块元素(可设置宽高，不独立成行)：img，input，button。

display属性：

  • bolck：块元素。
  • inline：行内元素。
  • inline：行内块元素。
  • none：隐藏元素。(list-style: none可以隐藏序号)

Css定位(position)

  1. >绝对定位：absolute(脱离文档流)
  2. >相对定位：relativr(不脱离文档流)
  3. >固定定位：fixed(脱离文档流-视窗)
  4. >坐标属性(非定位元素不起作用) ->top/left/right/bottom/z-index

形变(transfrom)

| 整个盒子模型内所有一起形变(以下为常用)

  1. rotate：旋转；其deg表示角度>>>[例旋转45°<顺时针>：transform: rotate(45deg);]
  2. scale：缩放>>>[例缩放1.5倍：transform: scale(1.5);]
  3. translate：位移>>>[例X+50px;Y+100px：transform: transla(50px,100px);]
  4. 同时使用例子：transform: scale(1.5) rotate(45deg);(中间空格)

过渡效果(transition)

  • 简写：[transition: 属性 秒数 函数 延迟,属性 秒数 函数 延迟, ......]
  • 过渡的属性>>>[例：transition-property:Name;]
  • 过渡的时间>>>[例：transition-duration: 1s;]
  • 过渡函数[transition-timing-function]
    1. ease：开始&结束慢，中间快<默认>
    2. linear：匀速
    3. ease-in：开始慢
    4. ease-out：结束慢
    5. ease-in-out：和ease类似，但幅度更大
  • 过渡的延迟[例：transition-delay: 1s;]
  • 设置其原点[例：transfrom-origin: X Y;<默认0 0>]
